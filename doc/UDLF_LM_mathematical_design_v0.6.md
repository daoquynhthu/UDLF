# 统一扩散潜场语言模型：严格数学设计 v0.6


## v0.6 修订说明

本版只处理第 7 节训练目标中的方向性与权重定义问题，不增加新的架构机制：

1. 明确区分：
   - 期望条件对数似然是边缘对数似然的下界；
   - 其负值 \(\mathcal L_t^{\mathrm{prior}}\) 是真实边缘 NLL 的上界。
2. 补充多样本先验目标的稳定实现与路径梯度权重：
   \[
   \widetilde w_r
   =
   \operatorname{softmax}_r(\ell_r).
   \]
   多路径梯度不是均匀平均，而是由各路径对真实 token 的相对似然加权。
3. 重写 posterior dropout 下的总损失：
   - 先验项始终保持固定权重 \(\lambda_p\)；
   - 后验项只在 \(m_t=1\) 时启用；
   - 默认使用逆保留概率归一化，避免 \(p_{\mathrm{drop}}\) 退火改变后验项的期望权重。
4. 规定 \(0\le p_{\mathrm{drop}}<1\)，并限制最大 dropout，防止归一化导致方差失控。
5. 在训练伪代码中明确多样本 `logsumexp` 计算和 posterior dropout 的跳过逻辑。
6. 在稳定性监测章节明确：有限时间 Lyapunov 指数此处用于数值稳定性诊断，定义与记忆分析统一引用第 13.5 节。

## 0. 设计目标与约束

本架构暂称 **统一扩散潜场语言模型**（Unified Diffusive Latent-Field Language Model，UDLF-LM）。

它试图验证以下核心命题：

> 一个因果文本模型能否只依赖单一、持续存在的连续潜在场，在统一非线性状态耦合、状态依赖耗散和状态依赖扩散的共同作用下，自然形成宽广、连续、上下文相关的有效时间尺度谱，并通过标准反向传播完成训练。

设计中明确排除：

- 人为划分的快状态、中状态、慢状态；
- world state、self state、controller state 等预命名状态；
- MoE 或专家路由；
- 第一版中的竞争固定点、侧向抑制和赢家选择；
- 显式时间尺度监督；
- 在线学习、STDP、eligibility trace；
- 多套针对不同时间跨度的预测头；
- 依赖无限长 token KV cache 的旁路记忆；
- 把扩散限制为局部文本块解码器。

训练允许标准反向传播，但监督信号只评价文本概率，不直接指定内部时间组织。

## 0.1 方法谱系与研究边界

UDLF-LM 不宣称发明一种全新的连续随机动力学门类。它与以下既有范式存在直接联系：

- **Neural ODE / Latent ODE**：使用参数化连续动力系统演化隐藏状态；
- **ODE-RNN 与事件驱动潜变量模型**：观测到来时执行离散状态跳变，观测之间连续演化；
- **Latent SDE**：同时学习状态依赖漂移、扩散及变分路径后验；
- **selective SSM / Mamba 类模型**：根据输入选择性传播、写入和遗忘持续状态。

UDLF-LM 的研究对象不是上述单个数学部件，而是它们在逐 token 因果语言建模中的特定组合：

1. 固定容量的持续潜在场作为历史压缩状态；
2. 离散 token 通过跳变算子注入状态；
3. token 事件之间执行共享非线性随机动力学；
4. 每个事件边界进行自回归词表读出；
5. 不预设离散时间尺度或人工记忆层级；
6. 通过跨 token 状态干预检验潜在状态的真实因果作用。

因此，潜在贡献应被表述为语言建模问题设定、动力机制组合和可证伪实验体系，而不是把观测跳变、状态依赖扩散或变分路径推断本身声称为独有结构。

# 1. 时间、观测与状态

## 1.1 外部离散时间

文本序列记为

\[
x_{0:T}=(x_0,x_1,\ldots,x_T),
\qquad
x_t\in\{1,\ldots,|\mathcal V|\},
\]

其中 \(\mathcal V\) 是词表。

外部时间 \(t\) 表示 token 事件顺序。模型在吸收 \(x_t\) 后预测 \(x_{t+1}\)。

## 1.2 内部连续时间

在两个相邻 token 事件之间，引入归一化内部时间

\[
s\in[0,1].
\]

这个区间不是一个人为定义的认知时间尺度，只是连续动力学的积分区间。数值求解时可以离散为若干微步，但微步数只是求解精度参数。

## 1.3 持续潜在场

模型唯一的长期内部状态为

\[
Z_t(s)\in\mathbb R^{M\times d},
\]

其中：

- \(M\)：潜在单元或潜在位置数量；
- \(d\)：每个潜在单元的通道维度；
- \(Z_t(0)\)：在吸收 token \(x_t\) 后的初始潜在场；
- \(Z_t(1)\)：演化到预测 \(x_{t+1}\) 时的终端潜在场。

潜在场不会在每个 token 后重新初始化，而是持续传递：

\[
Z_{t+1}(0)
=
\mathcal J_\theta
\left(
Z_t(1), e(x_{t+1})
\right),
\]

其中 \(\mathcal J_\theta\) 是观测注入算子，\(e(x)\) 是 token 嵌入。

因此所有远期历史只能通过持续潜在场影响未来预测。

状态持续仅适用于同一因果流，例如同一文档、同一会话或被明确声明连续的事件流。对于统计上独立的训练样本或无关文档，默认执行：

\[
Z_{\mathrm{start}}
=
Z_{\mathrm{init}},
\]

其中 \(Z_{\mathrm{init}}\) 是固定零状态或可学习初始状态。不得无条件把上一独立文档的终端状态传入下一文档，否则会引入样本间泄漏、训练顺序依赖和不可复现性。

训练分块边界与因果流边界必须区分：

- 同一因果流的分块：状态数值继续传递，可选择截断梯度；
- 独立样本边界：状态重置；
- 显式跨会话记忆：属于未来策略层，不是第一版默认行为。

# 2. 因果生成过程

完整因果过程为：

\[
Z_0(0)\sim p_0,
\]

对每个 \(t=0,\ldots,T-1\)：

1. 从 \(Z_t(0)\) 出发，在内部时间 \(s\in[0,1]\) 上演化；
2. 得到终端状态 \(Z_t(1)\)；
3. 从 \(Z_t(1)\) 读出下一个 token 分布；
4. 采样或观察 \(x_{t+1}\)；
5. 将 \(x_{t+1}\) 注入潜在场，形成 \(Z_{t+1}(0)\)。

即：

\[
Z_t(0)
\longrightarrow
Z_t(1)
\longrightarrow
p_\theta(x_{t+1}\mid Z_t(1))
\longrightarrow
Z_{t+1}(0).
\]

严格因果条件为：

\[
p_\theta(x_{t+1})
=
p_\theta(x_{t+1}\mid x_{\le t}),
\]

未来 token 不得进入先验动力学或读出路径。

# 3. 观测注入算子

## 3.1 Token 嵌入

设

\[
e_t
=
E[x_t]
\in
\mathbb R^{d_e},
\]

其中 \(E\in\mathbb R^{|\mathcal V|\times d_e}\) 为可学习词嵌入矩阵。

## 3.2 潜在场对观测的选择性吸收

对潜在单元 \(i=1,\ldots,M\)，定义：

\[
q_i
=
W_q\,\operatorname{RMSNorm}(Z_i),
\qquad
k_t
=
W_k e_t.
\]

观测分配权重为：

\[
\alpha_i
=
\frac{
\exp(q_i^\top k_t/\sqrt d)
}{
\sum_{j=1}^M
\exp(q_j^\top k_t/\sqrt d)
}.
\]

候选观测写入为：

\[
\Delta_i
=
\tanh
\left(
W_v e_t + W_z Z_i
\right).
\]

状态依赖写入门为：

\[
\eta_i
=
\sigma
\left(
W_\eta[
\operatorname{RMSNorm}(Z_i);
e_t
]
\right).
\]

最终观测跳变：

\[
\mathcal J_\theta(Z,e_t)_i
=
\operatorname{RMSNorm}
\left(
Z_i+\alpha_i\,\eta_i\odot\Delta_i
\right).
\]

该算子只定义新观测如何扰动当前场，不规定信息应保存多久。

## 3.3 注入算子的稳定性与捷径监测

观测注入每个 token 只发生一次；内部积分步数 \(K\) 增加并不会提高观测跳变的频率。因此不预设“跳变与小步长必然导致高频梯度振荡”。

真正需要监测的是注入映射对既有状态的覆盖和梯度放大。

定义相对跳变幅度：

\[
R_{\mathrm{jump},t}
=
\frac{
\|\mathcal J_\theta(Z_t,e_t)-Z_t\|_{\mathrm{rms}}
}{
\|Z_t\|_{\mathrm{rms}}+\varepsilon
}.
\]

定义注入 Jacobian：

\[
J_{\mathcal J,t}
=
\frac{
\partial\mathcal J_\theta(Z_t,e_t)
}{
\partial Z_t
}.
\]

使用 power iteration 或 Jacobian-vector product 估计：

\[
s_{\max,t}
\approx
\|J_{\mathcal J,t}\|_2.
\]

还应测量：

- 写入门 \(\eta_i\) 的均值、分位数和饱和比例；
- 分配权重 \(\alpha_i\) 的熵；
- 注入前后状态余弦相似度；
- 注入后旧信息保持率；
- 去除内部动力学、只保留注入算子时的语言建模性能。

若只靠注入算子就能完成大部分预测，说明模型可能绕过持续动力学。

第一版不默认对 \(\mathcal J_\theta\) 施加强谱约束。只有在观察到：

- \(s_{\max,t}\) 长期显著大于 1；
- 状态跳变反复覆盖既有信息；
- 梯度爆炸与注入事件高度相关；

时，才测试：

- 写入门上限；
- 谱归一化；
- Jacobian penalty；
- 局部梯度裁剪。

这些措施必须作为显式消融，避免稳定约束过早压制必要的状态重组。

# 4. 统一非线性潜在交互核心

第一版不使用竞争固定点、稀疏赢家选择或侧向抑制。所有潜在单元通过同一个共享非线性核心参与状态演化。

## 4.1 归一化潜在表示

对每个潜在单元定义：

\[
R_i
=
\operatorname{RMSNorm}(Z_i).
\]

RMSNorm 只用于构造漂移与扩散网络的输入，不直接覆盖或重置原始状态 \(Z_i\)。

## 4.2 潜在单元交互

潜在单元之间通过共享多头注意力交换信息：

\[
C
=
\operatorname{MHA}(R,R,R),
\qquad
C_i\in\mathbb R^d.
\]

将当前因果条件投影为：

\[
u_t\in\mathbb R^d.
\]

定义门控 MLP 的中间维度：

\[
d_{\mathrm{ff}}
=
r_{\mathrm{ff}}d.
\]

默认取：

\[
r_{\mathrm{ff}}=4.
\]

于是：

\[
W_u,W_v
\in
\mathbb R^{d_{\mathrm{ff}}\times 3d},
\]

\[
U_i
=
W_u[R_i;C_i;u_t]+b_u
\in
\mathbb R^{d_{\mathrm{ff}}},
\]

\[
V_i
=
W_v[R_i;C_i;u_t]+b_v
\in
\mathbb R^{d_{\mathrm{ff}}}.
\]

采用门控非线性：

\[
H_i
=
\operatorname{SiLU}(U_i)\odot\sigma(V_i)
\in
\mathbb R^{d_{\mathrm{ff}}}.
\]

最后：

\[
W_o
\in
\mathbb R^{d\times d_{\mathrm{ff}}},
\]

\[
Y_i
=
W_oH_i+b_o
\in
\mathbb R^d.
\]

这里不存在预设的快通道、慢通道、稀疏组或专家身份。所有时间行为必须从统一状态动力学中形成。

## 4.3 第一版的简约性要求

第一版不额外加入：

- 通道级稀疏惩罚；
- Top-\(K\) 选择；
- sparsemax 或 entmax；
- 侧向抑制矩阵；
- 活动疲劳；
- 竞争固定点；
- 稀疏路由器。

这些机制只有在基础模型已经证明持续场、状态依赖耗散和扩散具有独立价值后，才能作为后续扩展单独检验。

# 5. 持续潜在场的随机动力学

## 5.1 先验随机微分方程

在吸收 \(x_t\) 后，潜在场按以下 SDE 演化：

\[
dZ_t(s)
=
f_\theta
\left(
Z_t(s),e_t
\right)\,ds
+
G_\theta
\left(
Z_t(s),e_t
\right)\,dW_s,
\qquad
s\in[0,1].
\]

其中：

- \(f_\theta\)：状态依赖的确定性漂移；
- \(G_\theta\)：状态依赖的扩散算子；
- \(W_s\)：标准 Wiener 过程。

扩散作用于持续潜在场本身，而不是一个局部文本块解码器。

## 5.2 确定性漂移

由统一非线性潜在交互核心得到 \(Y\) 后，定义状态依赖耗散：

\[
\Lambda_i
=
\lambda_{\max}
\sigma
\left(
W_\lambda[R_i;Y_i]
\right),
\]

其中

\[
0\le \Lambda_i\le\lambda_{\max}.
\]

漂移为：

\[
f_{\theta,i}(Z,e)
=
\beta_{\max}\tanh(Y_i)
-
\Lambda_i\odot Z_i.
\]

这里没有固定的“快维度”或“慢维度”。

局部动力学结构可由完整 Jacobian 描述：

\[
J(Z,e)
=
\frac{\partial f_\theta(Z,e)}{\partial Z}.
\]

这里 \(f_\theta\) 内部使用的 RMSNorm、注意力、门控非线性、状态依赖耗散和原始 \(Z\) 路径都必须一并微分。

在某一轨迹邻域内，若局部模态特征值满足：

\[
\operatorname{Re}\lambda_j<0,
\]

可以定义局部线性化松弛时间：

\[
\tau_j^{\mathrm{local}}
=
-\frac{1}{\operatorname{Re}\lambda_j}.
\]

它只描述当前状态邻域内的小扰动响应，不能直接等同于跨 token 的全局记忆长度。强非线性吸引子、状态切换、循环轨道和可重新激活结构，都可能使全局记忆行为与瞬时谱显著不同。

因此局部谱只作为机制诊断；真正的记忆时间尺度必须主要通过跨 token 脉冲响应、状态干预和行为保持曲线测量。

## 5.3 状态依赖扩散

训练型随机模型要求扩散严格非退化。对每个潜在单元和通道定义：

\[
\Sigma_i
=
\varepsilon_\sigma
+
(\sigma_{\max}-\varepsilon_\sigma)
\sigma
\left(
W_\sigma[R_i;Y_i]
\right),
\]

其中：

\[
0
<
\varepsilon_\sigma
<
\sigma_{\max}.
\]

采用对角扩散：

\[
G_\theta(Z,e)
=
\operatorname{Diag}
\left(
\operatorname{vec}(\Sigma)
\right).
\]

因此：

\[
dZ_i
=
f_{\theta,i}\,ds
+
\Sigma_i\odot dW_i.
\]

扩散强度由当前状态、潜在单元交互和当前因果输入共同决定，但没有额外监督规定哪些方向应更随机。

完全确定性的模型不通过令训练型 \(\Sigma\) 逼近零获得，而作为独立 ODE 消融定义：

\[
G_\theta
\equiv
0.
\]

这样可以把严格随机路径模型和确定性持续场模型在数学与实验上明确分开。

## 5.4 扩散的语义解释边界

状态依赖扩散首先只是一个**随机潜在动力学机制**。在没有额外证据前，不得直接把它解释为：

- 认知不确定性；
- 偶然不确定性；
- 多模态历史信念；
- 探索欲望；
- 对多个语义解释的显式维护。

扩散可能吸收多种效应：

- 固定容量压缩造成的信息损失；
- 文本本身的多义性；
- 模型容量不足；
- 参数估计误差；
- 分布外输入；
- 正则化与鲁棒化；
- 优化过程中有益的随机扰动；
- 后验和先验不匹配。

控制能量：

\[
\frac12
\int
\|c_\phi(s)\|_2^2ds
\]

只表示在当前先验动力学下，为使路径更符合已观察目标所需的白化漂移修正。它不能单独被称为认知不确定性。

只有当以下干预证据同时成立时，才允许把扩散进一步解释为历史状态不确定性：

1. 保持局部 token 统计近似相同，只改变远期历史的可辨识程度；
2. 历史越含糊，先验路径分散度越高；
3. 后续消歧证据到来后，路径分布选择性收敛；
4. 路径簇与可解释语义分支对应；
5. 这种状态分布结构相对 ODE、输出 softmax 和确定性 ensemble 提供额外预测或校准收益。

否则，即使扩散有性能收益，也只能称为有效的随机潜在动力学，不能赋予更强的认知语义。

# 6. Token 读出

## 6.1 条件化多头潜在场汇聚

固定单查询并非固定权重，因为注意力权重仍依赖 \(Z_i\)，但它只提供一个固定的相关性方向。第一版采用少量共享多头查询，而不引入专家化读出。

根据当前观测和潜在场平均状态生成读出条件：

\[
r_t
=
W_c
\left[
e_t;
\frac1M\sum_i Z_i
\right].
\]

对 \(h=1,\ldots,N_r\)，定义条件化查询：

\[
q_{t,h}
=
q_h^{(0)}+W_{q,h}r_t.
\]

每个读出头的注意力权重为：

\[
\omega_{i,h}
=
\frac{
\exp(q_{t,h}^\top W_{r,h} Z_i/\sqrt d)
}{
\sum_j
\exp(q_{t,h}^\top W_{r,h} Z_j/\sqrt d)
}.
\]

头表示：

\[
h_{t,h}
=
\sum_i\omega_{i,h}Z_t(1)_i.
\]

最终汇聚：

\[
h_t
=
W_{\mathrm{merge}}
\operatorname{Concat}
(h_{t,1},\ldots,h_{t,N_r}).
\]

所有头共享同一个词表输出层，不形成独立专家。默认 \(N_r=4\)，并与单查询版本做消融。

## 6.2 词表分布

读出 logits：

\[
\ell_t
=
E_{\text{out}}W_{\text{out}}
\operatorname{RMSNorm}(h_t)+b.
\]

可令输出嵌入与输入嵌入绑定：

\[
E_{\text{out}}=E.
\]

下一个 token 分布为：

\[
p_\theta(x_{t+1}\mid x_{\le t})
=
\operatorname{softmax}(\ell_t).
\]

模型没有直接访问无限历史 token；所有历史影响必须通过 \(Z_t\) 保留。

# 7. 随机路径训练：分阶段设计

第一版必须把“扩散是否有价值”和“受控后验是否有效”分开验证。扩散属于核心研究线，但受控变分后验不作为最初实现的前置条件。

## 7.1 先验随机路径

生成与第一阶段训练均使用：

\[
dZ
=
f_\theta(Z,e_t)\,ds
+
G_\theta(Z,e_t)\,dW_s.
\]

它只能依赖过去、当前 token 和当前潜在状态。

## 7.2 第一阶段：直接先验路径目标

设一条先验随机路径的终端状态为 \(Z_t^{p}(1)\)。定义：

\[
\mathcal L_t^{\mathrm{prior}}
=
-
\mathbb E_{p_\theta(Z_t^p(\cdot)\mid Z_t(0),e_t)}
\log
p_\theta
\left(
x_{t+1}\mid Z_t^p(1)
\right).
\]

边缘预测概率为：

\[
p_\theta
\left(
x_{t+1}\mid Z_t(0),e_t
\right)
=
\mathbb E_{p_\theta(Z_t^p(\cdot)\mid Z_t(0),e_t)}
p_\theta
\left(
x_{t+1}\mid Z_t^p(1)
\right).
\]

由 Jensen 不等式：

\[
\mathbb E_{p_\theta}
\log
p_\theta
\left(
x_{t+1}\mid Z_t^p(1)
\right)
\le
\log
p_\theta
\left(
x_{t+1}\mid Z_t(0),e_t
\right).
\]

因此必须明确区分两个方向：

\[
-\mathcal L_t^{\mathrm{prior}}
\]

是边缘对数似然的下界，而：

\[
\mathcal L_t^{\mathrm{prior}}
\ge
-
\log
p_\theta
\left(
x_{t+1}\mid Z_t(0),e_t
\right)
\]

是边缘负对数似然的上界。

所以，最小化 \(\mathcal L_t^{\mathrm{prior}}\) 是有效但可能较松的训练目标。文档后续若使用“先验路径下界”，专指 \(-\mathcal L_t^{\mathrm{prior}}\) 对边缘对数似然的下界，不把正损失本身称为下界。

可以使用 \(S\) 条重参数化先验路径形成多样本下界。令：

\[
\ell_r
=
\log
p_\theta
\left(
x_{t+1}\mid Z_t^{p,(r)}(1)
\right).
\]

定义：

\[
\mathcal L_{t,S}^{\mathrm{prior}}
=
-
\mathbb E
\left[
\log
\left(
\frac1S
\sum_{r=1}^{S}
\exp(\ell_r)
\right)
\right].
\]

数值实现必须使用稳定的 log-sum-exp：

\[
\mathcal L_{t,S}^{\mathrm{prior}}
=
-
\mathbb E
\left[
\operatorname{LSE}(\ell_1,\ldots,\ell_S)
-
\log S
\right].
\]

对固定的重参数化噪声样本，其梯度为：

\[
\nabla_\theta
\mathcal L_{t,S}^{\mathrm{prior}}
=
-
\sum_{r=1}^{S}
\widetilde w_r
\nabla_\theta \ell_r,
\]

其中：

\[
\widetilde w_r
=
\frac{\exp(\ell_r)}
{\sum_{j=1}^{S}\exp(\ell_j)}
=
\operatorname{softmax}_r(\ell).
\]

这里的 \(\nabla_\theta\ell_r\) 包括：

- 读出网络的直接梯度；
- 通过 \(Z_t^{p,(r)}(1)\) 的路径梯度；
- 通过重参数化噪声轨迹进入漂移和扩散参数的梯度。

因此，多样本目标不是 \(S\) 条路径 NLL 的简单算术平均。似然较高的路径在当前样本梯度中获得更大权重。直接实现为：

```text
log_prob_r = log_softmax(logits_r)[target]
loss = -(logsumexp(log_prob_r, dim=sample) - log(S))
```

即可由自动微分获得正确权重。

第一阶段的目的只是确认：

- 扩散参数能够获得稳定梯度；
- 随机路径不导致状态崩溃；
- 状态依赖扩散相对固定扩散和 ODE 基线是否有初步收益。

## 7.3 第二阶段：受控训练后验

只有第一阶段稳定后，才允许后验控制器读取真实目标 token \(x_{t+1}\)：

\[
dZ^q
=
\left[
f_\theta(Z^q,e_t)
+
G_\theta(Z^q,e_t)c_\phi(Z^q,e_t,e_{t+1},s)
\right]ds
+
G_\theta(Z^q,e_t)dW_s^{(q)}.
\]

其中 \(c_\phi\) 只存在于训练后验中，不参与生成。

后验控制器不是时间尺度控制器。它只用于推断哪些随机潜路径更能解释实际出现的下一个 token。

## 7.4 双轨因果状态协议

后验读取 \(x_{t+1}\)，因此其终端状态包含生成时不可获得的目标条件。即使 \(x_{t+1}\) 在下一步已成为合法历史，直接把后验终端状态持续传递仍会造成明显的训练—生成制度差异，并可能重复编码目标信息。

默认采用双轨协议。

### 先验轨迹

\[
Z_t^p(0)
\longrightarrow
Z_t^p(1).
\]

它负责：

- 当前 token 的纯先验预测；
- 跨 token 状态持续；
- 与生成阶段完全一致的状态分布。

下一 token 的持续状态只能由先验轨迹和真实观测注入形成：

\[
Z_{t+1}^p(0)
=
\mathcal J_\theta
\left(
Z_t^p(1),e_{t+1}
\right).
\]

### 后验轨迹

后验轨迹与先验轨迹从同一个起点开始：

\[
Z_t^q(0)
=
Z_t^p(0).
\]

它只用于：

- 当前 token 的变分重构项；
- 路径 KL；
- 先验—后验局部对齐；
- 分析目标条件对潜在轨迹的影响。

默认禁止：

\[
Z_{t+1}^p(0)
=
\mathcal J_\theta
\left(
Z_t^q(1),e_{t+1}
\right).
\]

即后验终端状态不得直接成为下一 token 的持续状态。

### 可选 filtering-posterior 消融

可以单独测试把后验状态传播到下一步的 filtering posterior 版本，但必须标记为不同训练制度，并与双轨默认方案比较：

- teacher-forced NLL；
- pure-prior rollout NLL；
- 长序列状态漂移；
- posterior–prior gap；
- 生成阶段崩溃程度。

任何依赖后验状态传播才能工作的模型，都不能被视为纯先验生成模型已经学会相同动力学。

## 7.5 连续路径的测度条件与控制能量

先验与后验共享相同扩散，漂移差参数化为：

\[
\Delta b
=
G_\theta c_\phi.
\]

因此 \(c_\phi\) 已位于白化 Brownian 控制空间。若直接参数化漂移差 \(\Delta b\)，才需要：

\[
c_\phi
=
G_\theta^{-1}\Delta b.
\]

在以下条件下：

1. \(G_\theta\) 严格非退化；
2. 先验和后验共享相同扩散；
3. 系数满足适当的可测性、Lipschitz 和增长条件；
4. 控制满足 Novikov 条件，例如

\[
\mathbb E
\exp
\left(
\frac12\int_0^1\|c_\phi(s)\|_2^2ds
\right)
<\infty;
\]

连续路径测度相对熵为：

\[
\operatorname{KL}(q_\phi\|p_\theta)
=
\frac12
\mathbb E_{q_\phi}
\int_0^1
\|c_\phi(s)\|_2^2ds.
\]

v0.5 通过 \(\varepsilon_\sigma>0\) 避免训练型扩散退化导致的测度歧义。

## 7.6 第二阶段单步 ELBO 与固定先验权重

受控后验的单步负 ELBO 为：

\[
\mathcal L_t^{(q)}
=
-
\mathbb E_{q_\phi}
\log
p_\theta
\left(
x_{t+1}\mid Z_t^q(1)
\right)
+
\frac{\beta}{2}
\mathbb E_{q_\phi}
\int_0^1
\|c_\phi(s)\|_2^2ds.
\]

当 \(\beta=1\) 且使用精确路径 KL 时，它对应标准变分下界的负值。其他 \(\beta\) 值属于 \(\beta\)-VAE 风格正则化目标，不再严格等同于原始边缘似然下界。

无论 posterior dropout 是否启用，先验轨迹项都必须保持固定权重：

\[
\lambda_p
\mathcal L_t^{\mathrm{prior}},
\qquad
\lambda_p>0.
\]

这样可以保证跨 token 持续的先验状态始终获得稳定训练信号。

## 7.7 Posterior dropout 与权重归一化

对每个 token 采样：

\[
m_t
\sim
\operatorname{Bernoulli}
\left(
1-p_{\mathrm{drop}}
\right),
\]

其中：

\[
0
\le
p_{\mathrm{drop}}
<
1.
\]

当 \(m_t=1\) 时，计算受控后验轨迹和 \(\mathcal L_t^{(q)}\)；当 \(m_t=0\) 时，直接跳过后验轨迹，不把退化后的后验重构项再次计入先验损失。

默认采用逆保留概率归一化：

\[
\mathcal L_t
=
\lambda_p
\mathcal L_t^{\mathrm{prior}}
+
\lambda_q
\frac{m_t}
{1-p_{\mathrm{drop}}}
\mathcal L_t^{(q)},
\qquad
\lambda_q\ge 0.
\]

对 dropout 随机变量取期望：

\[
\mathbb E_{m_t}
\left[
\lambda_q
\frac{m_t}
{1-p_{\mathrm{drop}}}
\mathcal L_t^{(q)}
\right]
=
\lambda_q
\mathbb E
\left[
\mathcal L_t^{(q)}
\mid
m_t=1
\right].
\]

因此，在 \(\mathcal L_t^{(q)}\) 的条件分布近似稳定时，改变 \(p_{\mathrm{drop}}\) 不会仅因采样频率变化而系统改变后验项的期望系数；先验项的系数始终固定为 \(\lambda_p\)。

必须注意，逆概率归一化会增加梯度方差。当：

\[
p_{\mathrm{drop}}
\to
1
\]

时，系数 \(1/(1-p_{\mathrm{drop}})\) 发散。因此第一版规定：

\[
p_{\mathrm{drop}}
\le
p_{\max}
<
1,
\]

例如：

\[
p_{\max}
=
0.8.
\]

并监测：

- 后验项梯度方差；
- 梯度裁剪触发率；
- 有效后验样本数；
- 不同 \(p_{\mathrm{drop}}\) 下的 prior/posterior loss 比例。

若逆概率归一化导致不可接受的方差，可以使用显式双批次估计：每个更新同时包含固定数量的先验样本和后验样本，再分别乘以 \(\lambda_p,\lambda_q\)。该方案计算更贵，但权重最清晰。

如果研究者有意让后验作用随训练逐渐减弱，可以不使用逆概率归一化，而采用：

\[
\mathcal L_t^{\mathrm{annealed}}
=
\lambda_p
\mathcal L_t^{\mathrm{prior}}
+
\lambda_q m_t
\mathcal L_t^{(q)}.
\]

此时必须明确承认：

\[
\mathbb E
\left[
\lambda_qm_t\mathcal L_t^{(q)}
\right]
=
\lambda_q
(1-p_{\mathrm{drop}})
\mathbb E
\left[
\mathcal L_t^{(q)}
\mid m_t=1
\right],
\]

即后验项的期望权重会随 dropout 日程下降。这属于有意的后验退火，不得与“仅防止后验捷径”混为一谈。

posterior dropout 只影响当前 token 的后验训练项，不改变双轨协议，也不改变先验状态的跨 token 传播。

## 7.8 先验—后验分布诊断

先验和后验不应被要求完全相同。后验读取了目标信息，理论上应更集中、更符合已观察 token。需要控制的是信息差距和覆盖关系，而不是消灭差距。

由于终端状态通常是高维隐式分布，不默认使用难以可靠估计的对称 KL。建议从同一 \(Z_t(0),e_t\) 采样多条先验和后验轨迹，测量：

- energy distance；
- MMD；
- sliced Wasserstein distance；
- classifier two-sample accuracy；
- 均值与协方差差异；
- 先验和后验输出 token 分布 KL；
- pure-prior rollout NLL；
- 后验控制能量。

若维度较高，优先使用 sliced Wasserstein、energy distance 和 classifier two-sample test。

期望现象不是距离趋近零，而是：

- 先验 NLL 随训练改善；
- 后验提供的信息增益受控；
- 先验对后验高概率状态区域具有非忽略覆盖；
- posterior–prior gap 不持续扩大；
- pure-prior rollout 不依赖后验状态传播。

完整训练段目标为：

\[
\mathcal L_{\mathrm{seq}}
=
\sum_{t=0}^{T-1}
\mathcal L_t.
\]

# 8. 离散数值实现与连续解释

第一版使用 Euler–Maruyama 离散化。

设每个 token 间隔使用 \(K\) 个内部积分微步：

\[
\Delta s
=
\frac1K.
\]

Euler–Maruyama 在通常正则条件下具有全局强收敛阶 \(1/2\) 和全局弱收敛阶 \(1\)。当 \(K=2\) 或 \(4\) 时，模型首先应被理解为带乘性噪声的共享随机残差递归，不能仅凭形式宣称已经逼近连续 SDE。

默认使用 \(K=4\)，并执行：

\[
K\in\{2,4,8,16\}
\]

消融与步长一致性检验。

## 8.1 先验微步

\[
Z^{k+1}
=
Z^k
+
f_\theta(Z^k,e_t)\Delta s
+
G_\theta(Z^k,e_t)
\sqrt{\Delta s}\,\epsilon_k,
\]

其中：

\[
\epsilon_k
\sim
\mathcal N(0,I).
\]

相应条件转移为：

\[
p_\theta(Z^{k+1}\mid Z^k)
=
\mathcal N
\left(
Z^k+f_\theta^k\Delta s,\;
G_\theta^kG_\theta^{k\top}\Delta s
\right).
\]

## 8.2 后验微步

\[
Z^{k+1}
=
Z^k
+
\left[
f_\theta(Z^k,e_t)
+
G_\theta(Z^k,e_t)c_\phi^k
\right]\Delta s
+
G_\theta(Z^k,e_t)
\sqrt{\Delta s}\,\epsilon_k.
\]

相应条件转移为：

\[
q_\phi(Z^{k+1}\mid Z^k)
=
\mathcal N
\left(
Z^k+
(f_\theta^k+G_\theta^kc_\phi^k)\Delta s,\;
G_\theta^kG_\theta^{k\top}\Delta s
\right).
\]

## 8.3 有限 \(K\) 下的精确离散路径 KL

在同一个当前状态 \(Z^k\) 上，先验和后验条件转移共享相同协方差。其均值差为：

\[
\Delta\mu_k
=
G_\theta^kc_\phi^k\Delta s.
\]

因此条件高斯 KL 精确为：

\[
\begin{aligned}
\operatorname{KL}
\left(
q_\phi(Z^{k+1}\mid Z^k)
\|
p_\theta(Z^{k+1}\mid Z^k)
\right)
&=
\frac12
\Delta\mu_k^\top
\left(
G_\theta^kG_\theta^{k\top}\Delta s
\right)^{-1}
\Delta\mu_k
\\
&=
\frac12
\|c_\phi^k\|_2^2\Delta s.
\end{aligned}
\]

由马尔可夫链 KL 的链式分解，有限 \(K\) 离散路径的 KL 为：

\[
\operatorname{KL}
\left(
q_\phi(Z^{1:K}\mid Z^0)
\|
p_\theta(Z^{1:K}\mid Z^0)
\right)
=
\frac12
\mathbb E_{q_\phi}
\sum_{k=0}^{K-1}
\|c_\phi^k\|_2^2\Delta s.
\]

因此，对**离散随机潜变量递归**而言，上式不是启发式 Riemann 近似，而是共享协方差参数化下的精确路径 KL。

## 8.4 连续 ELBO 与离散 ELBO 的关系

必须区分：

### 离散模型解释

有限 \(K\) 的 Euler–Maruyama 转移本身定义一个离散潜变量生成模型。使用第 8.3 节的精确离散路径 KL，可以得到该离散模型的严格变分下界。

### 连续模型解释

若把模型解释为连续 SDE，则有限 \(K\) 的轨迹分布、终端分布和控制能量只是连续对象的数值近似。连续解释只有在以下现象成立时才可信：

- 提高 \(K\) 后终端状态分布趋于稳定；
- token 输出分布收敛；
- 关键干预结论不依赖固定步长；
- 离散路径 KL 与连续控制能量估计趋于一致。

因此，第一版实现的首要数学身份是**离散随机潜变量递归**；连续 SDE 是需要通过数值一致性实验获得支持的极限解释。

\(K\) 是数值/递归深度参数，不代表模型内部存在 \(K\) 个时间尺度。

## 8.5 数值求解器优先级

第一版默认只实现 Euler–Maruyama，因为：

- 公式简单；
- 重参数化直接；
- 与精确离散高斯 KL 自然匹配；
- 计算成本最低；
- 当前首要问题是机制有效性而非高阶强路径精度。

只有当以下问题跨随机种子稳定出现时，才进入高阶求解器实验：

- 提高 \(K\) 后预测或轨迹不收敛；
- 强路径误差影响延迟歧义消解或扰动恢复结论；
- 扩散导数引起可识别的离散伪影；
- Euler–Maruyama 与更细步长参考解存在系统偏差。

候选顺序为：

1. stochastic Heun；
2. 在对角、可交换噪声条件满足时测试 Milstein；
3. 必要时再考虑更复杂随机求解器。

Milstein 需要扩散对状态的导数项，可能显著增加自动微分和显存成本，因此不作为第一版稳定性增强手段。高阶求解器只能作为离散化诊断，不能用来掩盖动力学本身不稳定。


# 9. 训练中的反向传播

## 9.1 反向路径

梯度通过以下路径传播：

\[
\text{token loss}
\rightarrow
\text{readout}
\rightarrow
Z_t(1)
\rightarrow
\text{SDE 微步}
\rightarrow
\text{统一非线性潜在交互核心}
\rightarrow
\text{漂移与扩散参数}
\rightarrow
Z_t(0).
\]

梯度通过离散化后的 SDE 微步、注意力、门控非线性、状态依赖耗散和扩散网络进行标准反向传播。

## 9.2 截断反向传播

潜在场跨整个文档持续传递，但为控制显存，采用随机长度截断反向传播。

每个训练段长度：

\[
L\sim\mathcal D_L,
\]

其中 \(\mathcal D_L\) 取宽分布，例如对数均匀分布，而不是固定长度。

在截断边界处：

\[
Z_{\text{next}}
=
\operatorname{stopgrad}(Z_{\text{prev}}),
\]

但状态数值继续传递。

随机边界可以减少固定截断长度对内部动力学时间谱的人工塑形。

# 10. 稳定性约束与适用边界

内部稳定主要通过参数化缩小可行域，而不是加入时间尺度监督。以下约束不能被解释为完整系统的全局稳定性证明。

## 10.1 有界状态驱动

\[
\|\beta_{\max}\tanh(Y)\|_\infty
\le
\beta_{\max}.
\]

该约束限制瞬时非线性驱动的幅度，但不限制完整 Jacobian 的非对角耦合。

## 10.2 非负状态依赖耗散

\[
0
\le
\Lambda
\le
\lambda_{\max}.
\]

非负耗散防止显式对角项直接变成反阻尼，但当 \(\Lambda\) 接近零时，并不保证长期回归或平稳性。

## 10.3 有界且非退化的扩散

\[
0
<
\varepsilon_\sigma
\le
\Sigma
\le
\sigma_{\max}.
\]

它保证训练型随机模型的路径测度条件，但持续非零扩散也可能导致长期随机游走，不能被视为稳定性来源。

## 10.4 非局部耦合与局部不稳定

由于：

\[
C
=
\operatorname{MHA}(R,R,R),
\]

每个 \(Y_i\) 可依赖所有潜在单元。完整漂移 Jacobian 包含：

\[
\frac{\partial f_i}{\partial Z_j},
\qquad
i\neq j.
\]

这些非对角块可以产生正或负耦合，并允许：

\[
\max_j
\operatorname{Re}
\lambda_j
\left(
\frac{\partial f}{\partial Z}
\right)
>
0
\]

在局部轨迹区域出现。局部扩张不一定是错误，它可能参与状态切换和非线性计算；但它意味着有界驱动与非负对角耗散不能推出全局收缩。

## 10.5 可解析保证的范围

在网络映射满足适当的局部 Lipschitz 与线性增长条件、漂移和扩散有界的情况下，可以支持有限时间解存在和无有限时爆炸的常规条件性论证。

但当前设计不解析保证：

- 全局渐近稳定；
- 全局收缩；
- 唯一平稳分布；
- 长期均方有界；
- 所有 Lyapunov 指数为负。

若未来需要解析的长期有界性，可以研究外域单边耗散条件：

\[
\langle Z,f_\theta(Z,e)\rangle
\le
C-\alpha\|Z\|^2,
\qquad
\alpha>0,
\]

或对投影矩阵施加谱约束。但这些约束可能压缩近中性模态和长时间尺度，因此不作为第一版默认结构。

## 10.6 状态归一化与 Jacobian 一致性

在观测跳变后使用 RMSNorm；连续微步后不对完整状态进行硬投影，以免把轨迹强制限制在固定流形上。

漂移和扩散网络内部读取：

\[
R_i
=
\operatorname{RMSNorm}(Z_i).
\]

因此 RMSNorm 是 \(f_\theta\) 与 \(G_\theta\) 的组成部分，其导数必须包含在 Jacobian-vector product、局部谱和稳定性分析中。由于耗散项仍直接使用原始 \(Z_i\)，完整动力学并不等价于单纯球面投影。

可采用软范数裁剪：

\[
Z
\leftarrow
Z
\min
\left(
1,
\frac{R_{\max}}{\|Z\|_{\mathrm{rms}}+\varepsilon}
\right).
\]

启用裁剪可以直接控制数值状态范围，但会改变原始动力学，因此必须作为显式配置并与无裁剪模型比较。

## 10.7 必须监测的稳定性指标

训练和长轨迹测试至少记录：

- \(\|Z_t\|_{\mathrm{rms}}\) 的均值、分位数与最大值；
- 状态二阶矩随序列长度的增长；
- 漂移范数与扩散能量；
- 完整 Jacobian 的谱横坐标估计；
- 有限时间最大 Lyapunov 指数估计；此处仅作为数值稳定性诊断，定义与记忆行为分析统一见第 13.5 节；
- 观测注入相对跳变幅度 \(R_{\mathrm{jump},t}\)；
- 注入 Jacobian 最大奇异值估计；
- 梯度范数和梯度爆炸频率；
- 不同随机种子下的轨迹发散速度。

最大 Lyapunov 指数只能说明轨迹对微扰的敏感程度，不能单独等同于记忆质量。它必须与行为保持、扰动恢复和输出分布变化联合分析。

第一版不把 persistent homology 作为必需测量。若已经观察到明确的亚稳态、循环轨迹或状态簇，再考虑：

- recurrence plots；
- metastable clustering；
- 返回概率；
- Koopman / transfer operator 模态；
- persistent homology。

稳定性最终由解析边界、数值监测与跨长度实验共同判断，而不是由第 10.1–10.3 节单独保证。

# 12. 必需消融与任务体系

## 12.0 基础动力学基线

必须增加以下容量匹配基线：

1. 线性确定性持续状态：

\[
Z_{t+1}
=
AZ_t+Be_t;
\]

2. 线性随机状态空间：

\[
dZ
=
(AZ+Be_t)ds+\Sigma dW;
\]

3. 非线性稠密 ODE；
4. 完整非线性状态依赖 SDE。

另设：

- 参数量匹配 selective SSM / Mamba 类基线；
- 持久状态字节数匹配 selective SSM / Mamba 基线；
- 截断窗口 Transformer；
- 长 KV cache Transformer 作为精确历史性能上界；
- 确定性 ensemble，区分扩散收益与普通多模型采样收益。

这用于区分：

- 持续状态本身的贡献；
- 非线性动力学的贡献；
- 状态依赖耗散的贡献；
- 状态依赖扩散的贡献；
- 固定状态压缩与原始历史保存的差异。

## 12.1 持续场消融

- 完整持续场；
- 每个 token 重置 \(Z\)；
- 每个短段重置 \(Z\)；
- 历史状态随机置换。

检验长期信息是否真正通过潜在场传递。

## 12.2 状态依赖耗散消融

比较：

- 完整状态依赖 \(\Lambda_\theta(Z,e)\)；
- 固定可学习耗散向量；
- 全局标量耗散；
- 无显式耗散但保留有界漂移；
- selective SSM / Mamba 类输入依赖衰减。

这用于判断宽时间行为是否主要来自直接衰减参数化。

## 12.3 扩散消融

扩散是第一版核心研究机制，但仍必须接受反事实检验：

- 完整状态依赖扩散；
- 固定非零扩散；
- 各向同性状态依赖扩散；
- 对角各向异性状态依赖扩散；
- ODE 反事实基线 \(G_\theta\equiv0\)；
- 确定性 ensemble；
- 推理时使用均值路径；
- 推理时随机采样路径。

评估扩散对以下性质的贡献：

- 序列似然；
- 不确定性校准；
- 状态扰动恢复；
- 延迟歧义消解；
- 后续证据到来后的状态重组；
- 多样性是否对应语义分支，而非随机表面变化。

第一版不会因早期单次负结果直接删除扩散；但如果在多个预先定义任务和容量设置下均无稳定作用，则必须重新审视整个随机潜场假设。

## 12.4 训练推断机制消融

按实现阶段比较：

1. 单样本先验路径下界；
2. 多样本先验路径下界；
3. 双轨受控后验；
4. filtering-posterior 状态传播；
5. 不同 \(\beta\)；
6. 不同 posterior dropout 日程；
7. 训练后验存在但梯度不进入先验漂移；
8. 后验只读取压缩目标嵌入；
9. 后验读取完整目标 token 嵌入。

检验：

- 扩散是否在没有后验的情况下已获得作用；
- 受控后验是否提高边缘似然近似；
- 收益是否来自后验捷径；
- 后验状态传播是否造成训练—生成失配；
- 先验生成是否真正接近训练后验。

## 12.5 历史状态反事实

- 正确状态；
- 清零状态；
- 样本间交换状态；
- 同一样本时间错位状态；
- 局部随机扰动状态。

正确状态必须稳定优于错误状态。

## 12.6 状态容量消融

\[
M
\in
\{8,16,32,64\}.
\]

比较参数量匹配与非匹配两种设置，避免把状态容量收益误认成参数量收益。

## 12.7 数值积分消融

\[
K
\in
\{2,4,8,16\}.
\]

除最终性能外，测量不同 \(K\) 下：

- 轨迹终点差异；
- token 分布 KL；
- Jacobian 估计稳定性；
- 推理成本；
- 是否存在离散化锁定。

只有在 Euler–Maruyama 出现稳定离散伪影后，才测试 stochastic Heun 或 Milstein。

## 12.8 读出结构消融

比较：

- 单固定查询；
- 单条件化查询；
- 多条件化查询；
- 简单 mean pooling。

多查询读出仍共享统一词表头，不引入专家网络。

## 12.9 非线性交互核心消融

比较：

- 潜在自注意力 + 门控 MLP；
- 仅门控 MLP；
- 仅线性状态耦合；
- 低秩双线性交互；
- 参数量匹配的 selective SSM 核心。

这用于判断通用非线性潜在场是否必要，还是结构化线性状态传播已经足够。

## 12.10 核心任务：延迟歧义消解

构造一个早期存在多个合理隐变量解释、后期才出现消歧证据的因果序列。

设隐变量：

\[
h
\in
\{1,\ldots,H\}.
\]

早期前缀 \(x_{\le t_0}\) 对多个 \(h\) 近似等可能；在延迟 \(D\) 后，证据 token \(x_{t_0+D}\) 指明正确解释；最终目标依赖该解释。

任务应包含：

- 合成可控版本；
- 自然语言代词指代；
- 叙事实体身份；
- 事件时间线分支；
- 规则在后期被限定或推翻。

核心指标：

### 消歧前

- 路径状态簇数量与可分离度；
- 对各解释的预测概率；
- 早期过度承诺率；
- calibration error；
- 路径多样性与语义分支的一致性。

### 消歧时

- 正确解释概率提升速度；
- 错误解释概率衰减速度；
- 状态分布收缩时间；
- 状态移动距离；
- 控制能量或先验重组代价。

### 消歧后

- 最终任务准确率；
- pure-prior rollout NLL；
- 干扰后的解释恢复率；
- ODE、固定扩散、状态依赖扩散和确定性 ensemble 的差异。

只有随机路径在消歧前保留有意义的可分离分支，并在证据到来后选择性收敛，才能支持“历史解释不确定性”这一强解释。

## 12.11 任务类型分层

不得仅用综合 perplexity 判断架构成败。至少分为：

### A. 精确存储任务

- 随机字符串和数字回忆；
- 延迟复制；
- 长距离键值检索；
- 任意位置引用。

固定容量模型在这类任务上天然可能弱于长 KV cache。主要比较对象是固定状态 SSM 和截断 Transformer。

### B. 语义压缩任务

- 长文主题保持；
- 实体关系持续；
- 叙事状态跟踪；
- 长程论证一致性；
- 多次改写后的核心事实保持。

### C. 状态纠错任务

- 早期错误解释被后续证据推翻；
- 中途插入干扰后恢复原任务；
- 潜在状态受扰动后恢复；
- 规则发生条件化修订。

### D. 不确定性任务

- 延迟歧义消解；
- 多义词；
- 不完整上下文；
- 多种可能事件分支；
- 分布外输入；
- 数据噪声与知识缺失的区分。

架构核心价值主要应在 B、C、D 类任务中体现，而不是宣称全面替代精确历史缓存。

## 12.12 公平计算与内存预算

至少报告两类预算下的比较。

### 相同计算预算

匹配或记录：

- 训练总 FLOPs；
- 每 token FLOPs；
- wall-clock 训练时间；
- tokens/s；
- GPU 利用率；
- 推理延迟；
- 能耗估计。

### 相同持久内存预算

匹配：

- 潜在场字节数；
- SSM 状态字节数；
- KV cache 字节数；
- 截断窗口可保留的 token 数。

还应报告：

- 峰值训练显存；
- 批量大小敏感性；
- 序列长度扩展时的吞吐变化；
- 达到相同任务性能所需计算量；
- 每增加一单位长期语义性能所需的额外 FLOPs。

UDLF 若使用数倍计算只获得少量 perplexity 改善，不足以证明工程价值。它需要在状态纠错、延迟歧义消解、扰动恢复或非线性内容恢复上展示定性能力收益。

# 13. 涌现时间尺度与全局动力学测量

训练中不监督时间尺度，只在训练后测量。

记忆时间尺度的主要证据来自跨 token 干预与行为后果；局部 Jacobian 谱只用于解释机制，不能单独作为成功标准。

## 13.1 跨 token 脉冲响应与行为保持

在时刻 \(t\) 对潜在场施加受控小扰动：

\[
Z_t'
=
Z_t+\delta v.
\]

沿相同后续输入和相同随机种子传播，测量：

\[
D_Z(k)
=
\|Z_{t+k}'-Z_{t+k}\|,
\]

以及行为差异：

\[
D_p(k)
=
\operatorname{KL}
\left(
p_\theta(\cdot\mid Z_{t+k}')
\|
p_\theta(\cdot\mid Z_{t+k})
\right).
\]

真正有意义的持续性要求潜在差异在输出行为上产生可测后果，而不只是隐藏状态范数不同。

## 13.2 信息保持曲线

向文本中注入一次性符号或事实，在不同延迟后测试其对输出概率的影响。

测量：

\[
I(\Delta)
=
\Delta
\log p(x_{\mathrm{target}}\mid Z_{t+\Delta}).
\]

## 13.3 上下文依赖性

对同一扰动 \(v\)，在不同上下文状态 \(Z^{(a)}\)、\(Z^{(b)}\) 下测量衰减。

若时间尺度真正属于动力学模式，而非固定坐标，应有：

\[
D_a(k)
\neq
D_b(k).
\]

## 13.4 局部 Jacobian 谱

沿真实文本轨迹估计完整漂移 Jacobian：

\[
J_t
=
\frac{\partial f_\theta}{\partial Z}
\bigg|_{Z_t}.
\]

使用 Arnoldi、Lanczos 或 Jacobian-vector product 估计主模态。对稳定局部模态定义：

\[
\tau_j^{\mathrm{local}}(t)
=
-\frac{1}{\operatorname{Re}\lambda_j(t)}.
\]

该量只解释局部小扰动的松弛结构。成功标准不是单纯出现宽谱，而是：

- 局部谱随上下文变化；
- 谱变化能预测脉冲响应曲线的局部形态；
- 与长期依赖任务表现存在稳定关联；
- 对状态依赖耗散、扩散和线性基线的差异具有解释力。

## 13.5 有限时间 Lyapunov 指数

估计有限时间最大 Lyapunov 指数：

\[
\lambda_{\max}^{(T)}
=
\frac1T
\log
\frac{
\|\delta Z_T\|
}{
\|\delta Z_0\|
}.
\]

它用于识别局部扩张、接近混沌边缘或过度收缩，但不能单独作为记忆质量指标。

必须联合报告：

- \(\lambda_{\max}^{(T)}\)；
- \(D_Z(k)\)；
- \(D_p(k)\)；
- 任务性能；
- 状态范数增长。

## 13.6 先验路径分布与语义分支

对同一历史前缀采样多条先验轨迹，测量：

- 路径终端的 energy distance；
- MMD；
- sliced Wasserstein；
- 聚类稳定性；
- 路径簇与真实语义分支之间的互信息；
- 输出分布差异。

如果路径分散只表现为连续数值抖动，而不能对应可解释分支或改善预测，则不能宣称模型维护了多模态历史解释。

## 13.7 全局结构分析的优先级

第一版优先使用：

- recurrence plots；
- 状态访问频率；
- metastable clustering；
- 返回概率；
- Koopman / transfer operator 近似；
- 线索触发的状态恢复。

persistent homology 仅在已经出现明确循环、簇结构或复杂流形证据后作为辅助分析，不属于第一阶段必需指标。

# 14. 训练与生成算法

## 14.1 阶段 A：先验随机路径训练

```text
输入同一因果流中的 token 序列
在独立样本边界初始化先验潜在场 Z_p

对每个 token 位置 t：
    1. 将当前 token x_t 注入 Z_p
    2. 从同一 Z_p 起点采样 S 条重参数化先验路径
    3. 每条路径运行 K 个先验随机微步
    4. 得到每条路径对真实 x_{t+1} 的 log_prob_r
    5. 用 -(logsumexp(log_prob_r) - log(S)) 计算多样本目标
    6. 由自动微分产生 softmax(log_prob_r) 加权的路径梯度
    7. 选择预先规定的状态延续策略：
       - 默认 S=1 时直接延续该路径
       - S>1 时必须预注册代表路径、均值状态或单独传播路径
    8. 观察真实 x_{t+1}
    9. 将 x_{t+1} 注入选定的 Z_p_end，形成下一 token 的先验起点
    10. 在随机训练分块边界 stop-gradient
    11. 在真实独立样本边界重置状态

对训练段执行反向传播
```

注意：当 \(S>1\) 时，多样本目标只定义了当前 token 的边缘似然估计，并没有自动定义哪一条潜在路径应跨 token 延续。第一版默认 \(S=1\) 完成持续状态实验；多样本训练必须单独定义并消融状态延续规则，避免用似然最高路径造成隐式后验选择。

阶段 A 必须先完成以下验证：

- 随机轨迹训练稳定；
- 扩散参数没有统一贴近上下界；
- ODE、固定扩散和状态依赖扩散能够被清楚比较；
- 不同 \(K\) 的结论具有基本一致性；
- 延迟歧义消解任务中随机路径具有可测结构，而非纯表面噪声。

## 14.2 阶段 B：双轨受控后验训练

```text
从阶段 A 的可用先验模型开始
维护唯一跨 token 持续的先验状态 Z_p

对每个 token 位置 t：
    1. 将当前 token x_t 注入 Z_p，得到共同起点 Z_0
    2. 从 Z_0 运行 K 个先验微步，得到 Z_p_end
    3. 始终计算 lambda_p * prior_path_loss
    4. 采样 m_t ~ Bernoulli(1 - p_drop)
    5. 若 m_t = 1：
         a. 从同一个 Z_0 运行 K 个后验微步，得到 Z_q_end
         b. 计算 posterior reconstruction 与离散路径 KL
         c. 加入 lambda_q / (1 - p_drop) * posterior_loss
         d. 计算 prior/posterior 分布诊断指标
       若 m_t = 0：
         a. 跳过后验轨迹
         b. 不重复计入退化后的后验重构项
    6. 观察真实 x_{t+1}
    7. 只将 x_{t+1} 注入 Z_p_end，形成下一 token 的持续状态
    8. 丢弃 Z_q_end，不允许其跨 token 传播
    9. 在随机训练分块边界 stop-gradient

对训练段执行反向传播
```

若采用不归一化的后验退火目标，必须在实验配置中明确标记，并报告后验项实际期望权重：

\[
\lambda_q
(1-p_{\mathrm{drop}}).
\]

posterior dropout 只影响当前 token 的后验控制，不改变先验状态传播协议。

可以把 filtering-posterior 作为单独消融，但不得与默认双轨结果混合报告。

## 14.3 生成

```text
输入提示 token
逐 token 注入持续先验场

循环：
    1. 不使用后验控制
    2. 运行 K 个先验随机微步
    3. 从终端潜在场获得下一个 token 分布
    4. 采样或选择 token
    5. 将生成 token 注入潜在场
```

生成可有两种模式：

- 确定性诊断模式：令 Brownian 噪声为零，使用均值轨迹；
- 随机生成模式：正常采样 Brownian 噪声。

确定性诊断模式不是训练模型的严格边缘预测，只用于研究漂移场本身的行为。

# 15. 最小成功判据

架构进入下一阶段前，至少满足：

1. 相同参数量下，持续场模型在长延迟任务上优于无状态或重置状态基线；
2. 正确潜在状态显著优于清零、交换和错位状态；
3. 跨 token 脉冲响应和行为保持曲线呈现宽广、上下文相关的持续性；局部 Jacobian 谱只能作为辅助解释；
4. 完整非线性潜在场优于容量匹配的线性持续状态基线；
5. 状态依赖耗散相对固定耗散提供稳定收益，或至少产生明确不同的上下文相关记忆行为；
6. 状态依赖扩散相对固定扩散、ODE 和确定性 ensemble，在预注册的至少一类核心任务上提供可重复收益；
7. 延迟歧义消解中，随机路径在消歧前呈现与语义分支相关的状态结构，并在证据到来后选择性收敛；
8. 阶段 A 的先验随机路径模型本身可稳定训练，不依赖受控后验才能工作；
9. 若进入阶段 B，默认双轨协议下 pure-prior rollout 必须稳定，posterior–prior gap 不持续扩大；
10. filtering-posterior 若显著优于双轨协议，必须明确承认训练—生成制度差异，不能视为先验模型能力；
11. 潜在状态范数、二阶矩、随机积分和梯度在更长于训练截断长度的轨迹上保持可控；
12. 不依赖额外的时间尺度标签、状态类型标签或多头未来监督；
13. 在相同持久状态内存预算下，相对 selective SSM / Mamba 展现长期语义、纠错或歧义处理优势；
14. 若计算成本显著更高，必须展示定性能力收益，而非仅有小幅 perplexity 改善；
15. 对扩散的认知或多模态解释，只有在历史可辨识度干预、路径分支和后续收敛证据同时成立时才能提出。

# 16. 主要风险

## 16.1 扩散退化

若输出 softmax 已足以表达不确定性，模型可能把 \(\Sigma\) 压到 \(\varepsilon_\sigma\) 下界附近。

这不是数值错误，而可能意味着潜空间扩散没有独立价值。必须通过 ODE、固定扩散、状态依赖扩散和确定性 ensemble 的消融判断。

## 16.2 扩散语义不可识别

即使扩散改善性能，它也可能主要发挥：

- 正则化；
- 鲁棒化；
- 类似 dropout 的随机扰动；
- 优化逃逸；
- 输出多样性增强；

而非历史状态不确定性建模。

必须通过只改变历史可辨识度的干预、延迟歧义消解和路径语义分支分析区分这些解释。

## 16.3 后验捷径

训练后验读取真实 \(x_{t+1}\)，可能承担过多信息，使先验动力学学不到有效预测。

需要监测：

- 路径 KL；
- pure-prior rollout NLL；
- 先验与后验输出分布 KL；
- energy distance / MMD / sliced Wasserstein；
- classifier two-sample accuracy；
- posterior dropout；
- 先验对后验高概率状态区域的覆盖。

## 16.4 后验状态跨 token 泄漏与制度失配

若后验终端状态直接传入下一 token，训练时持续状态会依赖生成阶段不存在的目标条件，并可能重复编码 \(x_{t+1}\)。

默认双轨协议禁止后验状态传播。

若 filtering-posterior 版本表现更好，必须单独报告：

- teacher-forced NLL；
- pure-prior rollout NLL；
- 长序列生成崩溃；
- 状态分布偏移；
- 对后验轨迹的依赖程度。

## 16.5 Posterior dropout 权重漂移

若在 \(m_t=0\) 时仍把退化后验重构项计入损失，或不区分归一化 dropout 与有意后验退火，则先验/后验的有效梯度权重会随 \(p_{\mathrm{drop}}\) 日程变化。

默认使用固定先验项加逆保留概率归一化的后验项，并限制 \(p_{\mathrm{drop}}\le p_{\max}<1\)。必须报告后验项的实际梯度范数和有效采样率。

## 16.6 多样本路径的状态延续歧义

多样本先验目标定义当前 token 的边缘似然下界，但不自动决定哪条随机路径应跨 token 继续。

以下做法都改变模型含义：

- 延续最高似然路径：引入目标条件选择，近似隐式后验；
- 延续均值状态：可能落到低概率区域；
- 随机延续一条路径：保持生成一致性但方差较大；
- 同时维持多条路径：状态和计算成本随样本数增长。

因此第一版持续状态实验默认 \(S=1\)。任何 \(S>1\) 训练必须预注册状态延续规则并单独消融。

## 16.7 非线性核心冗余

潜在自注意力和门控 MLP 可能只实现一个昂贵的输入依赖衰减器，而没有形成真正不可由 selective SSM 表达的非线性状态结构。

必须通过线性、双线性、selective SSM 和完整非线性核心的容量匹配比较判断其必要性。

## 16.8 观测注入捷径

观测注入算子可能覆盖旧状态或直接承担大部分预测，使内部随机动力学退化为昂贵旁路。

需要比较：

- 完整模型；
- 只保留注入算子；
- 限制写入门；
- 关闭内部交互；
- 注入 Jacobian 与预测性能的关系。

## 16.9 训练速度

逐 token SDE 演化需要 \(K\) 次共享核心调用，无法像标准 Transformer 一样完全并行。

因此第一版应控制：

- \(M\)；
- \(d\)；
- \(K\)；
- 序列长度；
- 注意力复杂度。

## 16.10 时间尺度伪涌现

固定截断长度、固定求解步数或固定归一化结构，都可能制造表面上的时间常数。

必须随机化截断边界，并用上下文依赖扰动实验验证。

## 16.11 连续模型解释过度

有限 \(K\) 的实现首先是离散随机潜变量递归。若没有步长一致性、输出收敛和干预结论稳定性证据，就不能把连续 SDE 的性质直接归给实际模型。

离散 ELBO 可以严格成立，同时连续 SDE 解释仍然失败。两者必须分开报告。

## 16.12 离散化锁定

共享动力学核心可能学习专门适配固定 \(K\) 和固定步长的离散映射，而非近似连续向量场。

若提高求解精度后轨迹不收敛，必须把模型重新表述为离散随机递归，不得继续使用连续 SDE 的强解释。

## 16.13 高阶求解器误导

更高阶求解器可能降低数值误差，但不能修复：

- 错误的动力学假设；
- 不稳定漂移；
- 无用扩散；
- 后验捷径；
- 训练—生成失配。

Milstein 或 stochastic Heun 只能在已确认离散伪影后使用，不能被当作架构有效性的替代证据。

## 16.14 序列计算成本

模型每个 token 需要 \(K\) 次潜在场更新，并且潜在场跨 token 串行传递，计算复杂度和并行性显著弱于标准 Transformer 和可 parallel scan 的 selective SSM。

必须同时报告：

- 每 token FLOPs；
- 训练总 FLOPs；
- tokens/s；
- GPU 利用率；
- 推理延迟；
- 峰值显存；
- 持久状态字节数；
- 能耗估计；
- 达到相同性能所需计算量。

若机制收益不足以补偿顺序计算成本，架构即使数学成立也可能缺乏工程价值。

# 17. 核心定义

该架构可以严格定义为：

> 一个因果序列潜变量模型。其历史压缩状态是单一固定容量的持续潜在场；离散 token 通过跳变算子写入该场；在相邻 token 事件之间，潜在场按照由统一非线性交互核心参数化的状态依赖随机递归演化。该递归包含有界状态驱动、状态依赖耗散和严格非退化的状态依赖扩散。模型通过标准反向传播训练，但不对内部时间尺度、状态类别或记忆分工施加显式监督。

有限 \(K\) 时，模型的首要数学身份是离散随机潜变量马尔可夫链。其受控后验使用共享协方差高斯转移，因此路径控制能量给出精确离散路径 KL，并形成严格离散 ELBO。

只有当步长一致性实验支持状态轨迹、输出分布和因果干预结论随 \(K\) 收敛时，才进一步把该模型解释为事件驱动潜在 SDE 的数值实现。

状态只跨同一因果流和其训练分块持续；独立文档默认重置。

受控后验采用双轨协议：先验状态负责跨 token 持续，后验状态只用于当前 token 的局部变分训练。默认禁止后验终端状态直接传播到下一 token。

状态依赖扩散首先只被定义为随机潜在动力学。只有在历史可辨识度干预、延迟歧义消解、路径语义分支和后续选择性收敛均成立时，才允许进一步解释为历史状态不确定性建模。

内部时间组织的主要证据来自跨 token 因果干预和行为保持，而不是单独来自局部 Jacobian 谱。

# 18. 后续可选方向：竞争性稀疏动力学

竞争机制不属于第一版 UDLF-LM，不进入第一阶段实现、训练或核心结果归因。

## 18.1 进入竞争研究前必须满足的基础条件

只有在以下基础结论已经成立后，才考虑竞争机制：

1. 持续潜在场具有真实长期因果作用；
2. 状态依赖耗散已经完成与 selective SSM 的严格比较；
3. 状态依赖扩散已经证明具有独立价值；
4. 基础 UDLF-LM 在多个随机种子下稳定复现；
5. 存在可测量的冗余、贡献集中或跨模式干扰。

## 18.2 冗余与干扰的测量

不能只用单一阈值，例如“有效秩低于 \(M/2\)”决定是否引入竞争。至少测量以下四类证据。

### A. 表征有效秩

设潜在单元激活协方差为 \(C_Z\)，定义参与率形式的有效秩：

\[
r_{\mathrm{eff}}
=
\frac{
(\operatorname{tr}C_Z)^2
}{
\operatorname{tr}(C_Z^2)+\varepsilon
}.
\]

报告归一化值：

\[
\bar r_{\mathrm{eff}}
=
\frac{r_{\mathrm{eff}}}{M}.
\]

低有效秩可能表示冗余，也可能表示成功压缩，因此不能单独作为进入条件。

### B. 功能贡献集中度

逐个消融潜在单元，定义：

\[
\Delta_i
=
\mathcal L_{\mathrm{ablated}\ i}
-
\mathcal L_{\mathrm{full}}.
\]

对前 \(q\) 比例贡献最大的单元，定义集中度：

\[
C_q
=
\frac{
\sum_{i\in\operatorname{Top}_q}\max(\Delta_i,0)
}{
\sum_i\max(\Delta_i,0)+\varepsilon
}.
\]

若少量单元长期承担大部分可测贡献，才说明利用率明显集中。

### C. 稀疏替代曲线

只保留贡献最大的 \(qM\) 个单元或通道，测量：

\[
S(q)
=
\mathcal L_{\mathrm{keep}\ q}
-
\mathcal L_{\mathrm{full}}.
\]

只有在较小 \(q\) 下 \(S(q)\) 仍很小，才能说明稠密计算存在可替代浪费。

### D. 跨上下文干扰

对不同主题、任务或状态模式的梯度 \(g_a,g_b\)，测量：

\[
I_{ab}
=
\frac{
g_a^\top g_b
}{
\|g_a\|\,\|g_b\|+\varepsilon
}.
\]

持续显著的负相似度、单元覆盖冲突或状态更新互相破坏，可以构成引入动态分工机制的理由。

## 18.3 进入判据

竞争研究只有在预注册阈值下，至少出现以下联合证据时才启动：

- 表征重叠或低有效秩；
- 功能贡献高度集中；
- 稀疏替代损失很小；
- 跨上下文干扰显著。

结论必须：

- 跨至少三个随机种子复现；
- 在至少两个不同任务或数据分布上出现；
- 不能由减小模型规模、低秩化或普通门控轻易消除。

不要求四项全部成立，但不能仅凭单个有效秩指标启动竞争机制。

## 18.4 后续竞争版本

后续版本可以研究：

\[
a
=
\operatorname{ReLU}
\left(
s-\vartheta-\Gamma a
\right),
\]

并令竞争活动调制漂移与扩散。但必须作为独立增量实验，至少与以下机制比较：

- 稠密门控；
- Top-\(K\) 稀疏；
- entmax / sparsemax；
- 随机稀疏掩码；
- 低秩压缩；
- selective SSM；
- 无竞争的基础 UDLF-LM。

竞争机制只有在显著改变跨 token 行为保持、状态恢复、非线性制度切换或计算效率时，才能被视为有效结构；若只改变特征使用率，则应归类为普通稀疏路由。
